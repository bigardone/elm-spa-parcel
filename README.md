# Elm-spa Parcel template

Base project template using:

- **elm 0.19.1-3**
- **elm-spa 4.1.0**
- **elm-css 16.0.1**
- **Parcel 1.12.4**

## How to use this template

Just run:

- `git clone git@github.com:bigardone/elm-spa-parcel.git`
- `cd elm-spa-parcel`
- `npm start`

To build it:

- `npm run build`
